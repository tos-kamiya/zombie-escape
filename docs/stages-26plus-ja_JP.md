## 燃料ルールの違い（空の燃料缶が出るステージ）

空の燃料缶が登場するステージでは、燃料の流れが通常ステージと異なります。

- 通常の燃料缶ステージ: `プレイヤー -> 燃料缶 -> 車`
- 空の燃料缶が出現するステージ: `プレイヤー -> 空の燃料缶 -> 給油所 -> 車`

空の燃料缶が出現するステージでは、給油所を経由せずに脱出準備を完了することはできません。
